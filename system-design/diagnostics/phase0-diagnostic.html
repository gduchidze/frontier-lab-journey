Phase 0 Gate Exam

280 questions — 20 per topic across Track A (CS/systems), Track B (ML/DL/NLP), Track C (math). This IS the gate: STRONG topics get skipped entirely.

How the gates work One topic block (20 questions) = one sitting. Progress saves automatically. Verdict per topic the moment its 20th answer lands: STRONG (≥80% — topic SKIPPED, logged as a learning record) · SHAKY (50–79% — targeted mini-lesson on what you missed) · GAP (<50% — full lesson + primary-source reading before Phase deep work). Deep-dive links point at the canonical source for each topic. After each sitting, paste the summary block to your teacher.

Suggested order: one Track A topic, then one Track B, then one Track C per week — interleaving beats blocking for retention. Wrong answers are the deliverable.
