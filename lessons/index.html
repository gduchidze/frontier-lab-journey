How to use this course

  1. One lesson per 1–2 h session, in order — each builds on the last's cluster state.
  2. Do the labs for real. Reading YAML is not writing YAML.
  3. Answer quizzes from memory before clicking. Retrieval builds retention; recognition doesn't.
  4. When something breaks — good. describe, logs, then ask your teacher (the agent) with the output. Debugging is the curriculum.
  5. After each session, tell the agent what you did — it writes a learning record and calibrates the next lesson.
Spaced retrieval Start sessions 3+ with a 5-minute self-test: rebuild last lesson's key command/manifest from memory. The agent can quiz you — just ask "warm me up".
