მოწინავე ტრეკი — ნამდვილი პროდუქტიზაცია (მე-10 გაკვეთილის შემდეგ)

სენიორის დონის თეორია + ლაბორატორიები. მე-11 გაკვეთილში კლასტერს ხელახლა აშენებს Cilium-ითა და edge-პორტებით; მხოლოდ-GPU მექანიკა ისწავლება სიმულაციით, სიმძლავრის მათემატიკითა და დიზაინის დოკუმენტებით — პატიოსნად მონიშნული.

ფინალური ტრეკი — ატლასები (მე-18 გაკვეთილის შემდეგ)

ყველაფერი, რაც კურსმა არ მოიცვა, რუკაზე დატანილი: თემა → რა → როდის დაგჭირდება → სად წაიკითხო. თითოეული მთავრდება gap-map-ის თვითშეფასებით, რომელიც შენს მუდმივ გამეორების სილაბუსად იქცევა.

#გაკვეთილირითი გამოდიხარ
19Kubernetes-ის ატლასიK8s-ის სრული რუკა AI-ინჟინრებისთვის: Jobs/Kueue/LWS, StatefulSet-ები, PDB + graceful drain, operator-ები/CRD-ები, DRA, Kustomize, mesh, storage. ლაბორატორია: batch Job, ცოცხალი drain, kubectl debug, სათამაშო CRD, ხარვეზების რუკა
20vLLM-ის ატლასიvLLM-ის სრული რუკა: structured outputs, tool calling, მულტიმოდალურობა, multi-LoRA, offline batch, სწრაფი ჩატვირთვა, sleep mode, ეკოსისტემა (llm-d, Dynamo, SGLang). ლაბორატორია: API-ტური + offline Job + ხარვეზების რუკა

ცნობარები (ცოცხალი დოკუმენტები)

kubectl-ის შპარგალკაბრძანებები, pod-სტატუსების ამომხსნელი, კურსის ტერმინთა ლექსიკონი
მანიფესტის ანატომიაანოტირებული ჩონჩხები: Deployment, Service, PVC, probe-ები, ConfigMap/Secret
vLLM-ის შპარგალკაserve-ის flag-ები, CPU env-ცვლადები, OpenAI API-ის გამოძახებები, მეტრიკები
Helm-ის შპარგალკაბრძანებები, chart-ის სტრუქტურა, შაბლონების სინტაქსის გადარჩენის ნაკრები
Observability-ის შპარგალკაPromQL-პატერნები, vLLM-მეტრიკების ცხრილი, serving-მეტრიკების განმარტებები

როგორ გამოიყენო ეს კურსი

  1. თითო გაკვეთილი 1–2-საათიან სესიაზე, თანმიმდევრობით — თითოეული წინას კლასტერის მდგომარეობაზე შენდება.
  2. ლაბორატორიები რეალურად გააკეთე. YAML-ის კითხვა YAML-ის წერა არ არის.
  3. ქვიზებს მეხსიერებიდან უპასუხე, სანამ დააკლიკებ. გახსენება აშენებს დამახსოვრებას; ამოცნობა — არა.
  4. როცა რამე გატყდება — კარგია. describe, logs, შემდეგ ჰკითხე შენს მასწავლებელს (აგენტს) გამონატანთან ერთად. Debugging სწორედ სასწავლო პროგრამაა.
  5. ყოველი სესიის შემდეგ უთხარი აგენტს, რა გააკეთე — ის სწავლის ჩანაწერს წერს და შემდეგ გაკვეთილს აკალიბრებს.
ინტერვალური გახსენება მე-3 სესიიდან დაიწყე 5-წუთიანი თვითტესტით: მეხსიერებიდან აღადგინე წინა გაკვეთილის მთავარი ბრძანება/მანიფესტი. აგენტს შეუძლია გამოგკითხოს — უბრალოდ სთხოვე „გამახურე".
