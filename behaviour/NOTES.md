## Curriculum roadmap

### Phase A — Raw material (lessons 1–3)
1. **Story mining** — extract 10–15 candidate events from user's real history; bullet-outline top 8–10; map to competency matrix. ← lesson 0001
2. **Story upgrade to senior bar** — rewrite each story's Action/Result for organizational impact ("changed how the team ships" not "fixed the bug"); quantify results.
3. **The failure & conflict stories** — the hardest ones done properly: real ownership of mistakes, self-reflection, no hero-editing. These decide senior-level credibility.

### Phase B — Lab-specific modes (lessons 4–6)
4. **Anthropic culture round** — personal ethical framework articulation; live gray-area reasoning drills; reading: Core Views. Practice honest "I don't know / I was wrong".
5. **OpenAI mode** — STAR++ delivery, ownership/ambiguity bank rotation, AGI-safety engagement.
6. **DeepMind mode** — collaboration/feedback bank, research-culture literacy, mission specifics.

### Phase C — Execution (lessons 7+)
7. **Cross-loop story allocation** — which story to which round so nothing repeats across 4–6 interviews.
8. **Mock rotation** — 2+ human mocks; teacher runs drill sessions in-chat (random question, answer aloud, critique).

## Prep system (the mastery loop)
- Story bank lives in `reference/0001-story-bank-matrix.html` — bullets only, never scripts.
- Drill mode: user asks teacher "drill me" → teacher fires a random question from the per-lab banks, user answers out loud (types summary after), teacher critiques against that lab's grading column above.
- Every drill that surfaces a gap (missing story type, weak result quantification) → learning record.
